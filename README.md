# InSilico_Bench (Pro v14)

InSilico_Bench는 효모(Saccharomyces cerevisiae) 실험실에서 수행하는 주요 분자생물학 실험을 디지털로 시뮬레이션하고 관리할 수 있는 전문 도구입니다. 복잡한 PCR 설계, 상동 재조합(HR), 제한효소 처리 과정을 직관적인 UI와 SnapGene 스타일의 서열 뷰어를 통해 미리 확인하고 최적화할 수 있습니다.

## 최근 업데이트

### [v1.1.0] - 2026-05-12
*   **다중 생물종 지원**: 효모(SGD) 외에 인간, 마우스, 초파리 등 다양한 종의 유전자 검색 지원.
*   **프로바이더 시스템**: 종에 맞는 외부 데이터베이스(NCBI, PomBase 등) 자동 연결.
*   **로컬 서열 추출 강화**: GFF3/GBK 파일의 종 정보 관리 및 맞춤형 추출 로직.
*   **유연한 검색**: 로컬 파일이 없는 상태에서도 유전자 이름만으로 외부 DB 즉시 탐색 가능.

## 주요 기능

### 1. 🧬 PCR Execution (PCR 시뮬레이션)
*   **주형 및 프라이머 결합 분석**: 선택한 주형(Template)과 프라이머 쌍의 결합 위치 및 방향성을 자동 탐색합니다.
*   **SnapGene 스타일 시각화**: 주형 서열 위에 프라이머가 결합하는 위치와 방향을 화살표 트랙으로 표시합니다.
*   **결과 예측**: PCR 산물(Amplicon)의 예상 길이, 어닐링 온도(Ta), 신장 시간(Extension Time)을 자동으로 계산합니다.
*   **라이브러리 저장**: 생성된 Amplicon을 라이브러리에 저장하여 HR 시뮬레이션의 Insert로 즉시 활용 가능합니다.

### 2. 🔄 HR Simulation (상동 재조합 시뮬레이션)
*   **유전자 교체/삽입**: Target Genome과 Insert(Amplicon/Digest fragment) 간의 상동 재조합 결과를 예측합니다.
*   **자동 말단 탐색**: 양 끝단의 상동 영역(Homology Arm)을 자동으로 찾아 재조합된 최종 서열을 생성합니다.
*   **Feature 상속**: 재조합 과정에서 주석(Feature), 마커, 태그 정보가 유실되지 않고 최종 서열에 유지됩니다.

### 3. ✂️ Restriction Digest (제한효소 처리)
*   **REBASE 데이터베이스**: 수백 종의 제한효소 정보를 바탕으로 서열 내 Cut Site를 검색합니다.
*   **다중 효소 지원**: 여러 효소를 동시에 처리했을 때 생성되는 모든 단편(Fragments)을 목록화합니다.
*   **단편 미리보기**: 각 단편의 서열과 포함된 Feature를 SnapGene 뷰어로 즉시 확인할 수 있습니다.

### 4. 🧬 Gene Lookup (유전자 검색 및 추출)
*   **다중 종 지원 (Multi-species)**: 효모(SGD)뿐만 아니라 인간(NCBI), 마우스, PomBase, FlyBase, WormBase 등 다양한 종의 유전자 검색을 지원합니다.
*   **프로바이더 시스템**: 종별로 최적화된 외부 데이터베이스(NCBI Gene, SGD, PomBase 등)로의 자동 연결을 지원합니다.
*   **상/하류(Flanking) 추출**: 로컬에 로드된 GFF3/GBK 주석과 FASTA 서열을 매칭하여 유전자 및 주변 영역을 즉시 추출합니다.
*   **유연한 검색**: 로컬 데이터가 없는 경우에도 종을 선택하고 유전자 이름만 입력하면 즉시 해당 웹 데이터베이스 페이지로 연결됩니다.

### 5. 📁 Library Management (체계적인 라이브러리 관리)
*   **통합 관리**: 프라이머, 주형, 마커, 태그, 종(Species), 주석 참조 파일을 한곳에서 관리합니다.
*   **프라이머 검증**: 단일 프라이머의 이차 구조(Hairpin, Dimer) 분석 및 Tm 값을 계산합니다.
*   **가변 레이아웃**: 프라이머 목록 등 주요 테이블의 너비를 자유롭게 조절할 수 있습니다.

## 설치 및 실행 방법 (초보자용 가이드)

Python이나 프로그래밍을 잘 모르시는 분들도 아래 순서대로 따라하시면 프로그램을 실행할 수 있습니다.

### 1. Python 설치 (처음 한 번만 수행)
이 프로그램은 **Python 3.10** 이상의 환경이 필요합니다.
1.  [Python 공식 웹사이트](https://www.python.org/downloads/)에 접속합니다.
2.  노란색 **Download Python 3.xx.x** 버튼을 클릭하여 설치 파일을 다운로드합니다.
3.  설치 파일을 실행할 때 **반드시 하단의 "Add Python to PATH" 체크박스를 체크**하고 "Install Now"를 클릭하세요. (이 과정을 누락하면 명령어가 작동하지 않습니다.)

### 2. 프로그램 내려받기
1.  이 페이지 상단의 초록색 **[Code]** 버튼을 누르고 **[Download ZIP]**을 클릭합니다.
2.  다운로드된 압축 파일을 원하는 폴더에 해제합니다.

### 3. 필수 라이브러리 및 실행 (자동 배치 파일 이용)
프로그램 실행에 필요한 라이브러리 설치와 실행을 한 번에 처리해주는 배치 파일을 준비했습니다.

*   **Windows 사용자:** 
    1. 폴더 내에 있는 `InSilico_Bench.bat` 파일을 더블 클릭합니다.
    2. 처음 실행 시 필요한 라이브러리(Biopython, Primer3 등)를 자동으로 설치한 후 프로그램이 실행됩니다.
    3. 이후부터는 해당 파일을 더블 클릭하기만 하면 바로 프로그램이 뜹니다.

### 4. 수동 설치 및 실행 (터미널 이용 시)
배치 파일이 작동하지 않는 경우, 명령 프롬프트(cmd)에서 다음을 입력하세요.

```bash
# 1. 프로그램 폴더로 이동 (예시)
cd C:\path\to\InSilico_Bench

# 2. 필수 라이브러리 설치
pip install biopython pandas primer3-py openpyxl

# 3. 프로그램 실행
python main.py
```
